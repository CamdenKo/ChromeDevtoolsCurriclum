Inspect Element Tutorial

Getting started...

Google Chrome has lots of amazing tools that help developers speed up their development times drastically. This site will help walk you through using these tools. By the end of this, this site should be beautiful :')

Opening tools

To open dev tools you can click on the Settings on the top-right > Tools > Developer Tools
I highly recommend learning the keyboard shortcuts:

Devtools:
CTRL + SHIFT + I
CMD + OPT + I

Devtools > Console:
CTRL + SHIFT + J
CMD + OPT + J

Devtools > Inspect Element Mode:
CTRL + SHIFT + C
CMD + SHIFT + C

Take some time to get used to opening these tools quickly (it'll change your life).

Box-Model

Start by opening up the Devtools.
The devtools should be split into two parts: the "Elements" and the "Styles" tab.

The "Elements" tag should be fairly familiar - it's just HTML! Start by hovering over <html lang="en">. The entire page should be colored in blue! Next hover over <body class="container">. Interesting. There's no longer just blue, but therte's also a brown color on both the left and the right side. One great thing that Google Chrome does for us is split up elements by the Box-Model (Now's a good time to review this if you need) . So the acutal "Content" of the page is colored in

Blue

The tan color to the side is the

Margin

Click on the body tag and watch as the "Styles" half of the page inspector transforms.

Google does a great job here of describing the size of each aspect of the box-model. When you hover one part of the box-model, that part of the box model is highlighted.
