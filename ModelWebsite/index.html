Chrome DevTools Tutorial

Getting started...

Google Chrome has lots of amazing tools that help developers speed up their development times drastically. This site will help walk you through using these tools. By the end of this, this site should be beautiful :')

Opening tools

To open dev tools you can click on the Settings on the top-right > Tools > Developer Tools
I highly recommend learning the keyboard shortcuts:

Devtools:
CTRL + SHIFT + I
CMD + OPT + I

Devtools > Console:
CTRL + SHIFT + J
CMD + OPT + J

Devtools > Inspect Element Mode:
CTRL + SHIFT + C
CMD + SHIFT + C

Take some time to get used to opening these tools quickly (it'll change your life).

Navigating "Elements"

First make sure that you can see the "Elements" tab. You should be able to see the html of the page.

To navigate through, first click on an element. You can then use the Up/Down arrow keys to navigate through sibling elements. To expand/collapse a selection use the left/right arrow keys. Use this to try to find a div with ID "hidden" somewhere in the "Navigating Elements" section.

Another great tool that the Elements section gives us is underneath the HTML view. Right click on the above image and select "inspect". This should now select the img tag inside the Elements tab. You should now see "Img.marginVert10" in white text with a blue background.

This means that I inspected an image tag with the class "marginVert10" looking to the left of that are parent elements. With the oldest parent furthest to the left.

Editing CSS

Now that you are comfortable navigating through Chrome Developer Tools, now's your chance to actually develop. One of the greatest powers you gain is the edit to live edit CSS inside of the console. Let's start by centering the text below this.

Center Me!

Start by selecting the text in the Developer Tools console. You should now see that the text is surrounded by a div (this is done on purpose). The first thing we're going to learn is how to add existing classes to elements. Select the direct parent of the text, a div with the id "editMeFirst". Now, when you look above the box model you get plenty of different options. The option we are looking for is ".cls"

After clicking on ".cls", a new textbox that says "Add new class" will appear. Inside of that textbox type in "displayFlex" and hit enter to give that div the class, "displayFlex". Inside of the stylesheet I have already defined that class to be "display: flex".

Now let's add our own rule to the element. Click between the curly braces in "element.style" which should now allow you to type in a new rule. Let's modify the element's justify-content property. So type in justify-content and hit "Enter". This should then take you to the other side of the colon, now type in "center" and BLAM, the text is now centered.

Very cool. So we know how to add a class and how to edit one specific property on an element. However, we can also change the properties for an entire class!

Go back to the inspector and find a div with class marginVert10 and click on it.

Inside of the styles page we can now see that the rule "marginVert10" gives this element a margin-top and margin-bottom of 10 px each. Let's change that to 10. To do so, click on the "0px;". After doing so, that text should now be selected. We can now either type in 10px or use the arrow keys to increment/decrement the value one at a time. Try using the "Up" arrow on the arrow key to raise the margin-top and margin-bottom to 10 px each.

Every div that is marked "marginVert10" should have receieved margins at this point. That's because we edited the CSS rules for the class meaning that all elements with that class will have their styling changed with it.
